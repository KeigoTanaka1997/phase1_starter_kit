# Phase 1 Starter Kit (HTML/CSS/JS)

これは「AIに質問できる基礎」を身につけるための最小スターターです。

## 目的
- HTML/CSS/JSの最小構成でページを表示
- レスポンシブの基本を試す
- JSで軽いDOM操作（ボタンでテキスト切替）

## ローカルで開く
- `index.html` をブラウザで開くだけでOK（サーバ不要）

## 推奨拡張（VS Code）
- Prettier
- Live Server（拡張を入れて`index.html`を右クリック→Open with Live Server）
